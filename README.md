# 🤖 J.A.R.V.I.S. — (Demo)

> Demonstração interativa de uma solução de automação de processos (RPA) para agendamento hospitalar.

##  Sobre o projeto

O J.A.R.V.I.S. é um projeto desenvolvido para automatizar tarefas repetitivas do processo de agendamento ambulatorial, reduzindo o tempo gasto em atividades operacionais e minimizando erros manuais.

Esta aplicação é uma **demonstração interativa** da interface e do fluxo de execução da automação. Nenhum dado real é utilizado; todas as informações apresentadas são fictícias e foram criadas exclusivamente para fins de portfólio.

A versão completa da automação foi desenvolvida em Python e integra técnicas de RPA, OCR e automação web para executar o processo de forma automática.
